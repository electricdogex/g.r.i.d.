TronCode Quickstart
Type commands in the editor below and click Run.
  • GRID "Name" — rename the grid.
  • SPAWN x y [color] — add a node. Example: SPAWN 4 2 cyan.
  • LINK a b — connect node a to node b by id.
  • MOVE id x y — move a node.
  • DELETE id — delete a node.
  • FILL [color] — change background glow.
  • CLEAR — wipe nodes & links.
  • SAVE — persist immediately.
